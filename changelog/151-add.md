# Add prettier format check for notes

Can be enabled via config. `note.lint.prettier.config` accepts config options
from [Prettier Config](https://prettier.io/docs/en/options.html)

```yaml
note:
    links: ...
    styles: ...
    lint:
        prettier:
            enabled: true
            config:
                printWidth: 80
                useTabs: false
                tabWidth: 4
                proseWrap: always
```

Bad formatting can be fixed with `snage lint --fix`.
